About the Project

This internship task focuses on exploring Bootstrap 5 components, remixing them into original layouts, and producing polished pages that are responsive and accessible.

Goals
  • Learn Bootstrap core components
  • Design visually consistent pages
  • Deploy via GitHub Pages / Netlify
Design process

Start with examples, extract components (nav, hero, cards), remix with custom CSS utilities, and iterate on spacing & contrast to create a professional look.

Technical Notes

Tools

Bootstrap 5, HTML5, CSS3. Optionally deploy to GitHub Pages or Netlify.

Accessibility

Semantic HTML, contrast-aware color choices, and responsive behavior for assistive usage.

Customization

Utility classes keep the code clean; custom CSS used only for hero, glass card and hover effects.
